# Welcome to the farmer connect UI components

This project contains the UI components that are shared across multiple front end projects inside farmer connect.
The components are built with React and you can use this page to learn how to use or customize them.

These components are open-sourced under the [Apache License 2.0](https://github.com/farmerconnect/farmerconnect-ui/blob/main/LICENSE).

The code is hosted on farmer connect's [GitHub page](https://github.com/farmerconnect/farmerconnect-ui) and contributions can be made directly there. For that, just follow the project's README instructions.

The NPM package is hosted on [NPM registry](https://www.npmjs.com/package/@farmerconnect/farmerconnect-ui).
